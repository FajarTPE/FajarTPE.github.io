Kunshan University of Science and Technology (KSU)
Department of Information Engineering
Final Project: Remote Controlled Object Detection Vehicle with Web Integration.

During my Bachelor’s degree at Kunshan University of Science and Technology in the Department of Information Engineering, I embarked on an ambitious final project titled "Remote Controlled Object Detection Vehicle with Web Integration."

The aim of my thesis project was to explore and develop various methods for enabling computers to visually interpret the real world. This involved investigating solutions provided by the open-source computer vision library, OpenCV, and implementing these solutions in a practical application. The primary objective was to create a Raspberry Pi-based system capable of detecting and tracking objects in real time.

The implementation involved several key steps. First, I set up the Raspberry Pi with the necessary software, including the OpenCV library. Next, I configured the Arduino Uno to manage the vehicle's movement and sensory input. By establishing a communication link between the Arduino and Raspberry Pi, I enabled seamless data exchange for real-time processing. The final system was capable of detecting objects, tracking their movement, and providing feedback to the vehicle for navigation adjustments.

This project demonstrated the potential of integrating computer vision with embedded systems to create intelligent, responsive applications. The combination of OpenCV, Raspberry Pi, and Arduino Uno provided a robust platform for object detection and tracking, showcasing the practical applications of computer science and information engineering principles. The successful completion of this project highlighted the importance of interdisciplinary research and the practical implementation of theoretical knowledge.

2016 - 2019

Associate Degree (D3)

State Polytechnic of Tanah Laut (POLITALA)
Department of Informatics Engineering
Thesis Project Overview: Implementation of Single Sign On The State Polytechnic of Tanah Laut.

In the Informatics Engineering Study Program at the State Polytechnic of Tanah Laut, there are numerous web-based application systems that serve various functions. However, these systems were developed as standalone applications, requiring users to maintain separate accounts for each system. This often leads to users having to remember multiple usernames and passwords, increasing the likelihood of forgotten credentials and the practice of using easily guessable passwords. To address this issue, my thesis project focused on the implementation of a Single Sign-On (SSO) system, aimed at integrating and managing user authentication across all web-based applications.

The implementation of the SSO system significantly improved the user experience by allowing users to access multiple systems with a single set of credentials. This not only streamlined the login process but also enhanced security by reducing the reliance on easily guessable passwords. The integration of the Academic Information System and the Staff Information System demonstrated the effectiveness of the SSO system in managing authentication across different platforms.

The successful implementation of the Single Sign-On system at the State Polytechnic of Tanah Laut highlighted the benefits of integrated authentication solutions in educational institutions. By simplifying the login process and enhancing security, the SSO system addressed a critical need for both users and administrators. This project underscored the importance of user-friendly and secure authentication mechanisms in the digital age, paving the way for future enhancements and integrations.
